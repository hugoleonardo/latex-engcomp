	Para o ganho de $K = 161$, temos que:
		\[ G(s) = \frac{161}{(s+1)(s+2)(s+10)}\]
	
	Calculando $K_{p}$, utilizando a Equação \ref{eq:calculokp}:
	\begin{equation}
		K_{p} = \lim_{s \rightarrow 0} G(S)
		\label{eq:calculokp}
	\end{equation}		
	Temos:
		\[ K_{p} = \frac{161}{(1)(2)(10)} = 8,05\]
	Para o calcular o erro de regime utilizamos a Equação :
	\begin{equation}
		e(\infty) = \frac{1}{1+K_{p}}
		\label{eq:calculoerro}
	\end{equation}		
	Temos:
		\[ e(\infty) = \frac{1}{1+8,05} = 0,1105\]
		
	A Figura \ref{fig:questao1_sistema_nao_compensado} mostra o sistema não compensado.
	
	\begin{figure}[h!]
		\centering

		\caption{Sistema não compensado}		
		\label{fig:questao1_sistema_nao_compensado}
	\end{figure}
	
	O valor do zero escolhido arbitrariamente para o compensador PI (ideal) foi $Z_{c} = -0,2$, resultando na seguinte função de transferência para o compensador:
	\[ G_{c}(s) = \frac{K_{c}(s+0,2)}{s}\]
	Onde $K_{c}$ é o ganho do sistema compensado. As Figuras \ref{fig:questao1_LGR_Compensado} e \ref{fig:questao1Ganho_compensado} mostram respectivamente o novo LGR e o novo ganho ($K_{c} = 149$) do sistema.

\chapter{Questão 2}

	O sistema não compensado é o mesmo para a questão anterior, mostrado na Figura \ref{fig:questao1_sistema_nao_compensado} assim como o valor de $K_{p} = 8,05$ e o valor do erro $e(\infty) = 0,1105$. O erro de regime do sistema compensado é:
		\[ e_{c}(\infty) = \frac{e(\infty)}{10} = 0,011\]
	
	 Calculando o valor de $K_{pc}$:
		\[ K_{pc} = \frac{1 - e_{c}(\infty)}{e_{c}(\infty)} = 89,9\]
	Temos que:
		\[ \frac{Z_{c}}{P_{c}} = \frac{K_{pc}}{K_{p}} = \frac{89,9}{8,05} = 11,1677\]
	Escolhendo arbitrariamente $P_{c} = 0,05$, obtém-se $P_{c} = 11,1677*0,05 = 0,5584$. Com esses valores é possível montar a Função de transferência do compensador por atraso de fase, mostrada na Equação \ref{eq:questao2funcaocompensador}:
		\begin{equation}
			G_{c}(s) = \frac{K_{c}(s+0,5584)}{s+0,05}
			\label{eq:questao2funcaocompensador}
		\end{equation}

\chapter{Questão 5}
	
	A Figura   mostra o LGR e o ganho para o sistema não compensado.

	\begin{figure}[h!]

	Usando o valor de $T_{p}$ para calcular $\omega _{d}$ (parte imaginária do pólo):
		\[ T_{p} = \frac{\pi}{\omega _{d}} \rightarrow \omega _{d} = \frac{\pi}{1,047} = 3,0006 \cong 3\]
	A partir de $\omega _{d}$ encontra-se o valor de $\omega _{n}$:
		\[ \omega _{d} = \omega _{n}\sqrt{1-\xi ^{2}} \rightarrow \omega _{n} = \frac{3}{\sqrt{1-0,8^{2}}} \rightarrow \omega _{n} = 5\]
	Encontrando a parte real do pólo:
		\[ \sigma = \omega _{n}\xi = 5*0,8 = 4\]
	Usando a Equação \ref{eq:formulados} para encontrar o pólo dominante desejado:

		\begin{equation}
			s = -\omega _{n}\xi \pm \omega _{d}j
			\label{eq:formulados}
		\end{equation}			
		\[ s = -4 \pm 3j \]
	Os ângulos dos pólos do sistema estão representados no Apeêndice B. Abaixo será mostrado o cálculo do ângulo $\theta _{1}$ e do zero do controlador, $Z_{c}$.
		\[ \theta _{1} -\theta _{2} -\theta _{3} -\theta _{4} = 180 \]
		\[ \theta _{2} = tan^{-1}(\frac{3}{3}) = tan^{-1}(1) = 45^{o} \]
		\[ \theta _{3} = \theta _{4} = 90^{o} \]
		\[ \theta _{1} -45 -90 -90 = 180 \rightarrow \theta _{1} = 405^{o} = 45^{o} \]
	Calculando o valor de $Z_{c}$:
		\[ tan(45^{o}) = \frac{3}{Z_{c} - 4} \rightarrow Z_{c} = 7 \]
	A função de transferência do controlador PD é:
		\[ G_{PD}(s) = (s+7)\]				
	A função de transferência parcial do sitema com controlador PD é:
		\[ G(s) = \frac{2.99(s+7)}{(s+1)(s+4)}	\]
	As Figuras  mostras, respectivamente, o LGR e o ganho do sistema com o controlador PD, e a comparação das saídas do do sistema sem controlador (azul) e com controlador PD (verde). O tempo de pico do sistema com controlador PD foi $T_{p} = 0,8785$, bem próximo do valor desejado ($T_{p} = 1,047$).
