\chapter{Experimento 1}

	As Figuras \ref{fig:experimento1_LGR_original}, \ref{fig:experimento1_LGR_zero} e \ref{fig:experimento1_LGR_polo} mostram, respectivamente, o LGR de $G(s) = \frac{2}{s(s+1)}$, o LGR de $G(s) = \frac{2(s+2)}{s(s+1)}$e o LGR de $G(s) = \frac{2}{s(s+1)(s+2)}$. Quando um zero é adiconado ao LGR, o sistema permanece estável para qualquer valor positivo de $K$ ($K \geqslant 0$). Quando um pólo é adicionado ao LRG, conforme o servado abaixo, o LGR entra para o plano da instalbilidade (SPD), fazendo com que o valor de K possua uma faixa limitada de estabilidade que neste caso é $K \leqslant 3$.

\chapter{Experimento 3}

	Usando a tabela fornecida podemos encontra o valor de $R$ para $T_{s} = 3$. A Figura \ref{fig:experimento3_ganho} mostra o valor do ganho para o valor de $R$ encontrado. 
		\[ T_{s_{5\%}}  = -\frac{3}{R} \rightarrow R = -1 \]
	A Figura \ref{fig:experimento3_resposta} mostra a resposta do sistema simulado em malha fechada, o valor encontrado foi $T_{s} = 3,024$, confirmando o tempo de resposta desejado $T_{s_{desejado}} = 3$ .
